# **App Name**: AlgoRhythm Tickets

## Core Features:

- Event Listing: Display a list of upcoming university events with details such as date, time, location, and description.
- Ticket Purchasing: Allow users to purchase tickets online through a secure payment gateway.
- Ticket Management: Enable event organizers to manage ticket sales, track attendance, and generate reports.
- User Authentication: Secure user accounts for purchasing and managing tickets.
- Personalized Recommendations: AI powered tool which Recommends events to users based on their past purchase history, preferences, and profile information.

## Style Guidelines:

- Primary color: Deep red (#C0392B) to reflect the 'red color theme' as requested, creating a sense of excitement.
- Background color: Light, desaturated red (#F2D7D5) to provide a subtle backdrop that complements the primary color.
- Accent color: A slightly orange-shifted red (#D35400) is used to provide contrast and draw attention to key interactive elements.
- Headline font: 'Space Grotesk' sans-serif for headlines to give a computerized and scientific look, that suits the event name.
- Body font: 'Inter' sans-serif, a modern sans-serif that combines well with Space Grotesk.
- Use icons to visually represent event categories and ticket types.
- Maintain a clean and intuitive layout for easy navigation and ticket purchasing.
